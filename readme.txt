Send information to the specified e-mail address when the post published.

== Description ==
[View on GitHub.](https://github.com/marushu/post-notifier)

This plugin send to specified e-mail address you set.

=== What's sent? ===
* Post title
* Post thumbnail( If you set the Post Thumbnail )
* Post's URL

== Installation ==

1. Upload `post-notifier`directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Appear under Settings menu.
4. Set e-mail you want to send(comma-separated).
5. Select the post-type(multiple).
6. Set senter e-mail.
7. Click save.
